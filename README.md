# Predictiv Trend - AI-Powered Stock Analysis

Predictiv Trend is a modern, high-performance stock market analysis dashboard that leverages AI to provide intelligent insights onto stock performance. Built with speed and visual excellence in mind, it offers real-time streaming reports for any ticker combination over custom time ranges.

![Predictiv Trend Logo](./public/logo.svg)

## 🚀 Features

- **AI-Powered Insights**: Uses advanced language models (via Mistral AI) to analyze stock trends and provide qualitative reports.
- **Real-Time Streaming**: Reports are streamed directly to the UI, providing immediate feedback as the AI processes the data.
- **Dynamic Controls**: Easy-to-use ticker input and date range selection.
- **Premium Design**: Sleek, dark-themed UI featuring glassmorphism, smooth animations, and a responsive layout.
- **White-Labeled**: Fully customized branding with unique logos and professional metadata.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) + [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [TailwindCSS Animate](https://github.com/jamiebuilds/tailwindcss-animate)
- **Testing**: [Playwright](https://playwright.dev/) & [Vitest](https://vitest.dev/)

## 🏁 Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd predictiv-trend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create an optimized production build:
```bash
npm run build
```

## 🧪 Testing

The project includes both unit tests and end-to-end tests:

- **Unit Tests**: `npm run test`
- **E2E Tests**: `npx playwright test`
